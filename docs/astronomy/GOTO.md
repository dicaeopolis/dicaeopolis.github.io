# 关于（半）自动寻星与拍摄的一点思路

如果你不想买天文盒子/智能望远镜/谐波赤道仪这些玩意，那么下面这一套方法是一个比较简易合适，可玩性较高的拍摄方法，并且附加成本接近于 0。

## 第一步：对极轴

在容易看见北极星的夜晚，直接对准即可。

如果出于各种原因，比如遮挡、光污染等原因看不见北极星的话，可以考虑使用 [我这篇文章的方法](https://dicaeopolis.github.io/astronomy/polar/) 对极轴。原理就是基于地理纬度和地磁北极加上磁偏角修正来确定极轴方向。

需要注意的一点就是考虑到相机脚架和快装板等铁质设备可能会对磁场测定存在干扰，建议在不携带拍摄系统的时候测定好北极方向，并利用粉笔标注好。

## 第二步：找目标

对于 M42 和 M45 这种亮目标，或者 IC 434 这种亮星附近的目标，可以考虑直接目视对准。

但是如果背景里面没有亮星呢？如果是标准的德式赤道仪，直接在 Stellarium 里面查询目标的赤经赤纬，旋转到对应位置即可。这里对 Stellarium 的赤道仪构图功能狠狠点赞了。

我手头的是非标准的德式赤道仪（赤经轴有一个九十度的偏角），但是也可以用 [我这篇文章的方法](https://dicaeopolis.github.io/astronomy/track/) 的方法来找到目标。

## 第三步：实拍

一般而言对准之后就可以开始拍摄了，不过我们还可以验证一下构图。也就是对拍摄到的图片做星图解算。

利用我基于 Astrometry.net 做的 [这个项目](https://github.com/dicaeopolis/AstroLocal) 就能够实现本地化的高效解算。

极轴对准，构图完美，打开加热带，定好闹钟，就可以愉悦开拍了~

祝大家导星不拉线，夜夜是晴天~
